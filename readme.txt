Get Whos Amung Stats directly into the WordPress.

== Description ==

Show Whos Amung Stats in the Admin Bar, and in the Admin panel, without to go outside from your site.
== Installation ==

1. Upload the 'wp-whos-amung' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

= Usage =
1. Active the plugin.
2. Insert your Whos Amung ID in the Whos Amung configuration menu.
